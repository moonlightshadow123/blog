Nnet and RBFnet

Backprop on Nnet

Initiate all weights $w_{ij}^{(l)}$

  1. If stochastic: randomly pick $n \in {1,2, \dots, N}$.
  2. Forward: compute all $x_{i}^{(l)}$ with $x^{(0)} = x_n$.
  3. Backward: compute all $\sigma_j^{(l)}$ subject to $x^{(0)} = x_n$.
  4. Gradient descent: $w{ij}^{(l)} \leftarrow w{ij}^{(l)} - \eta x^{(l-1)}_j \sigma_j^{(l)}$

Return $g_{nnet}(x) = \tanh( \sum_j w_{jk}^{(2)} \cdot \tanh( \sum_i w_{ij}^{(1)} x_j))$

RBFnet

$$h(x) = output(\sum_{m=1}^M \beta_m RBF(x, \mu_m))$$

  • Full RBF M=N and $\mu_m = x_m$ 有多少数据就有多少样本中心，然后根据距离中心的远近投票。极端情况 — 只用一个投票： nearest neighbor.
  • SVM $$g_{svm}(x) = sign(\sum_{SV} \alpha_m y_m \exp(- r \Vert x - x_m \Vert^2 +b))$$
    只用 support vector 作为中心 $\mu_m$.

  • RBF network with k-means:

    1. run k-means with $K=M$ to get ${\mu_m}$.
    2. construct transform $\phi(x)$ from RBF(say, Gaussian) at $\mu_m$.
      $\phi(x) = [RBF(x, \mu_1), RBF(x, \mu_2), \dots, RBF(x, \mu_m)]$
    3. run linear model on ${(\phi(x_n), y_n)}$ to get $\beta$.
    4. return $g_{rbfnet}(x) = LinearHypothesis(\beta, \phi(x))$.

pLSA

$d$ — docoment; $c$ — topic; $w$ — word.
$$p(w \vert d) = p(d) \sum_c p(c \vert d) p(w \vert c)$$
$p(c \vert d)$ — multi-nominal distribution.
$p(w \vert c)$ — multi-nominal distribution.

LDA

$i$ — document; $k$ — topic; $v$ — word.
$\theta_{i} = [\theta_{i,1},\theta_{i,2},\dots, \theta_{i,K}]$ — document-topic distribution of document $i$.
$\phi_{k} = [\phi_{k,1}, \phi_{k,2}, \dots, \phi_{k,V}]$ — word-topic distribution of topic $k$.

Generative process of $w_{ij}$,

  1. Choose $\theta_i \sim Dir(\alpha)$, $i = 1, \dots, M$, $\theta_{i,k}$ = probability that document $i \in {1, \dots, M}$ has topic $k \in {1, \dots, K}$.
  2. Choose $\phi_k \sim Dir(\beta)$, $k = 1, \dots, K $, $\phi_{k,v}$ = probability of word $v \in {1, \dots, V}$ in topic $k \in {1,\dots, K}$.
  3. Choose $c_{ij} \sim Polynominal(\theta_i)$ (where $c_{ij} \in {1,\dots, K}$).
    Choose $w_{ij} \sim Polynominal(\phi_{c_{ij}})$ (where $w_{ij} \in {1, \dots, V}$).

PCA LDA

LDA (Linear Discrimination Analysis)

定义：
$E_{X \vert Y}[X \vert Y = i] = \mu_i$
$E_{X \vert Y}[(X - \mu_i)(X - \mu_i)^{-1}] = \Sigma_i$
$Z = w^T X$
The gist of LDA:
$$\max = \frac{\text{between class scatter}}{\text{within class scatter}} = \frac{\left(E_{Z \vert Y}[ Z \vert Y = 1] - E_{Z \vert Y}[ Z \vert Y = 0]\right)^2}{Var[Z \vert Y = 1] + Var[Z sss\vert Y = 0]}$$

其中：
$\left(E_{Z \vert Y}[Z \vert Y = 1] - E_{Z \vert Y}[Z \vert Y = 0]\right)^2 = (w^T(\mu_1 - \mu_0))^2 = w^T (\mu_1 - \mu_0)^T(\mu_1 - \mu_0) w$
$Var[Z \vert Y = i] = E_{Z \vert Y = i}[ \left(Z - \mu_i\right)^2 \vert Y = i] = w^T \Sigma_i w$
$$\begin{align*}
J(w) &= \frac{\left(E_{Z \vert Y}[Z \vert Y=1] - E_{Z \vert Y}[Z \vert Y = 0]\right)^2}{Var[Z \vert Y = 1] + Var[Z \vert Y=0]} \
&= \frac{w^T (\mu_1 - \mu_0)(\mu_1 -\mu_0)^T w}{w^T (\Sigma_1 + \Sigma_0)w} \
&= \frac{w^T S_B w}{w^T S_w w}
\end{align*}$$
其中：
$S_B = (\mu_1 - \mu_0)^T (\mu_1 = \mu_0)$ — between class scatter.
$S_w = \Sigma_1 + \Sigma_0$ — within class scatter.
解优化问题：
$$\max_w w^T S_B w\
s.t. w^T S_w w = k
$$
Language 函数：
$$L = w^T S_B w - \lambda (w^T S_w w - k)$$
对$w$求导得：
$$2 S_B w = 2 \lambda S_w w\
\implies S_w^{-1} S_B w = \lambda w
$$
可知$w$是$S_w^{-1} S_B$的特征值. 而$S_B = (\mu_1 - \mu_0)^T (\mu_1 - \mu_0)$是一个秩一矩阵，于是$S_w^{-1} S_B$唯一的特征值为$S_w^{(-1)} (\mu_1 - \mu_0)$

$A\alpha \alpha^T$的秩为1且唯一特征向量为$A\alpha$

于是$w^* = S_w^{-1}(\mu_1 - \mu_0) = (\Sigma_0 + \Sigma_1)^{-1} (\mu_1 - \mu_0)$

PCA (Principle Component Analysis)

$X = X - \bar{X}$ is 0 mean.
$\Sigma = X X^T$ is the covariance matrix.
Objective: transform $X$ to $AX$ with largest covariance. For a single vector $\alpha$.
$$\max \alpha X X^T \alpha^T\
s.t. \alpha^T \alpha = 1$$

So $\alpha$ is the eigenvector with the largest eigenvalue. For a matrix like $A$, we can choose top $k$ largest eigenvector.

PCA through SVD

SVD the matrix $X$:
$XX^T = C$, $D = PCP^T$ is a square diagonal matrix. Choose the top $k$ eigenvector in $P$ to be the $A$ matrix.

EM and k-means

EM algorithm

For each $i$, let $Q_i$ be some distribution over the $z$’s ($\sum_z Q_i(z) = 1, Q_i(z) \geq 0$)
$$\begin{align*}
\sum_i \log p(x^{(i)}; \theta) &= \sum_i \log \sum_{z^{(i)}} p(x^{(i)}, z^{(i)} ;\theta)\
&= \sum_i \log \sum_{z^{(i)}} Q_i(z^{(i)}) \frac{p(x^{(i)}, z^{(i)} ; \theta)}{Q_i(z^{(i)})}\
& \geq \sum_i \sum_{z^{(i)}} Q_i(z^{(i)})
\end{align*}$$
The last step of the above equation:

$$\log E_{z^{(i)}\sim Q_i} \left[\frac{p(x^{(i)}, z^{(i)} ; \theta)}{Q_i(z^{(i)})}\right] \geq E_{z^{(i)} \sim Q_i}\left[ \log \frac{p(x^{(i)}, z^{(i)} ; \theta)}{Q_i(z^{(i)}) } \right]$$

找到一个$Q_i$去极大化左边变成了找到一个$Q_i$去极大化右边，即极大化log期望的下限，当且仅当$\frac{p(x^{(i)}, z^{(i)}; \theta)}{Q_i(z^{(i)})}=c$ 时等号成立，所以 $ Q_i(z^{(i)}) \propto p(x^{(i)}, z^{(i)}; \theta)$

$$\begin{align*}
Q_i(z^{(i)}) &= \frac{p(x^{(i)}, z^{(i)}; \theta)}{\sum_z p(x^{(i)}, z^{(i)}; \theta)}\
&= \frac{p(x^{(i)}, z^{(i)}; \theta)}{p(x^{(i)}; \theta)}\
&= p(z^{(i)} \vert x^{(i)} ; \theta)
\end{align*}$$

So, 总结起来：
E-step:
$$Q_i(z^{(i)}) := p(z^{(i)} \vert x^{(i)}; \theta)$$
M-step:
Maximize log似然的下限：
$$\theta:= \arg\max_{\theta} \sum_i \sum_{z^{(i)}} Q_i(z^{(i)}) \log \frac{p(x^{i}, z^{(i)}; \theta)}{Q_i(z^{(i)})}$$

k-means

  1. 随机选取K个聚类质心点(cluster centroids)为$\mu_1, \mu_2, \dots, \mu_k \in R^n$.
  2. 重复下面过程直到收敛
    对于每个样例$i$, 计算其应该属于的类：
    $$c^{(i)} := \arg\min_j \Vert x^{(i)} -\mu_j \Vert^2$$
    对于每一个类$j$,重新计算该类的质心
    $$\mu_j = \frac{\sum^m_{i=1} I{ c^{(i)}=j } x^{(j)}}{\sum^m_{i=1} I{ c^{(i)} = j}}$$
