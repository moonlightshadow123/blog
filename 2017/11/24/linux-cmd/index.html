Linux Commands

Posted on 2017-11-24 | | Visitors

ls,cat,mkdir,rmdir,rm,cp,mv,less,touch,top,kill,ps,echo,chmod,whereis,useradd,userdel,groups,groupadd,groupdel,gpasswd,df,du,free,watch,head,tail,find,wc,cal,date,; && ||,ifconfig,tar,grep,netstat,ssh,scp,xargs,cut,awk,sed.

ls

显示目录内容列表

ls -l ———- 显示可见文件的详细信息。

ls -R ———- 递归显示.

cat

cat命令 连接文件并打印到标准输出设备上，cat经常用来显示文件的内容

cat > filename ———- 修改文件内容;
cat filename ———- 显示文件内容。

cat >> filename ———- append 文件内容。

cat file1 file2 >(or >>) file3 ———- 合并文件并存到 file3 中。

mkdir

用来创建目录.

`Create foldertestandtest1/test11(with parameter-por–parent`).

rmdir and rm

用于删除给定的文件和目录

rmdir -p dir1/dir2 ———- 删除指定的空目录。

rm -r dir1/dir2 ———- 递归删除指定的目录和其中的文件。

cp

将源文件或目录复制到目标文件或目录中

cp file1 file2 ... dir ———- 复制file1, file2, … 到dir中。

cp -r dir1 dir2 ———- 如果dir2不存在，则创建dir2并将dir1中的内容复制到dir2中； 如果dir2存在则将dir1整个复制到dir2中。

mv

用来对文件或目录重新命名

mv file1 file2 ———- 重命名文件。

mv file dir ———- 移动文件file到dir中。

mv dir1 dir2 ———- 如果dir2不存在，则将dir1重命名为dir2; 如果dir2存在则将dir1移动到dir2中。

less

分屏上下翻页浏览文件内容

less file ———- 打开翻页窗口. up arrow向上一行, down arrow向下一行, space翻页, q退出。

touch

创建新的空文件

touch file ———- 如果file不存在的话，创建新的空文件file; 如果file存在的话，只是修改文件的时间戳。

top

显示或管理执行中的程序

top ———- 显示当前进程. 参数 k + pid 终止某个进程; 参数 q 退出。

kill

删除执行中的程序或工作

kill -9 pid ———- 强制杀死pid的进程。

kill -l ———- 列出所有信号。

常用的信号：

ps

报告当前系统的进程状态

先用ps查找进程，然后用kill杀掉。
ps aux | grep xxx ———- 查找包含xxx的进程。

echo

输出指定的字符串或者变量

xxx="ooo" ———- 定义变量 xxx, 其值为 ooo.
echo $xxx ———- 打印变量 xxx 的值。

echo -e "string" ———- 参数-e允许转义。

chmod

用来变更文件或目录的权限

chmod u+x,o-w,a+r file ———- file对user的权限增加execute, 对other的权限减少write, 对ugo的权限全部增加read。

chmod u=rwx,g=rw,o=r file ———- file对user,对group,对other的权限分别设为 rwx, rw, r.

chmod 764 file ———- file对user，group，other的权限分别是rwx, rw, r. (r=4, w=2, x=1)

chmod user:market file ———- 将file给market组的user用户。

whereis

查找二进制程序、代码等相关文件路径

whereis java ———- 查找java的位置。

useradd

创建的新的系统用户

useradd user -g group ———- 将user加入到group中。

userdel

用于删除给定的用户以及与用户相关的文件

userdel username ———– 删除username用户。

userdel -r username ————- 删除username以及与该用户相关的全部文件。

groups

用来打印指定用户所属的工作组

groups username ———- 显示username所在的用户组。

groupadd

用于创建一个新的工作组

groupadd -g id groupname ———- 创建一个id为id, groupname为groupname的用户组。

groupdel

用于删除指定的工作组

groupdel groupname ———- 删除groupname为groupname的用户组。

gpasswd

Linux下工作组文件的管理工具

gpasswd -a username groupname ———- 将username添加到groupname中。

gpasswd -d username groupname ———- 将username从groupname中删除。

df

显示磁盘的相关信息

df -h ———- 以human readable的形式显示磁盘信息。

du

显示每个文件和目录的磁盘使用空间

du -h ———- 以human readable的形式显示每个文件和目录的空间使用情况。

free

显示内存的使用情况

free -h ———- 以human readable的形式显示内存的使用情况。

watch

可以将命令的输出结果输出到标准输出设备，多用于周期性执行命令/定时执行命令

watch -n interval cmd ———- 每interval秒执行一次cmd命令。

head

在屏幕上显示指定文件的开头若干行

head -num file ———- 显示file的前num行。

tail

在屏幕上显示指定文件的末尾若干行

tail -num file ———- 显示file的后num行。

find

在指定目录下查找文件

find path -name file ———– 在path中找到名为file的文件。

find path -regex exp ———- 在path中找到符合正则表达式exp的文件。

find path -type f -atime time ———- 在path中找到time之前被访问过的文件。

find path -type f -name file -exec cmd ;\ ———- 在path中找到名为file的文件，如果成功找到就执行cmd命令。

wc

统计文件的字节数、字数、行数

wc file ———- 统计file的行数，单词数，字符数。

wc -w file ———- 统计file的单词数。

ls -l | wc -l ———- 统计当前目录下的文件数。

cal

cal ———- 显示当前日历。

date

显示或设置系统时间与日期

date ———- 显示当前时间。

date -d "date" +"format" ———- 以format形式显示date的时间。

; && ||

执行多条命令的连接符号

cmd1; cmd2; cmd3 ———- 以;连接的命令无论执行的成功与否都会执行下一条。

cmd1 && cmd2 && cmd3 ———- 以&&连接的命令只有前一条成功执行才会执行下一条。

cmd1 || cmd2 || cmd3 ———- 以||连接的命令只有前面命令失败才会执行下一条。

ifconfig

配置和显示Linux系统网卡的网络参数

ifconfig eth0 up ———- 打开eth0网卡;
ifconfig eth0 down ———- 关闭eth0网卡。

ifconfig eth0 add 33ffe:3240:800:1005::2/64 ———– 给eth0添加33ffe:3240:800:1005::2/64的ipv6地址;
ifconfig eth0 del 33ffe:3240:800:1005::2/64 ———– 删除eth0的33ffe:3240:800:1005::2/64 ipv6地址。

tar

Linux下的归档使用工具，用来打包和备份。

tar -cvf name.tar file1 file2 ———- 把file1 file2打包成name.tar;
c for create, v for verbose, f for filename (f是最后一个参数，后面跟文件名).

tar -xvf name.tar ———- 把name.tar解压;
x for extraction, v for verbose, f for filename .

tar -czvf name.tar.gz file1 file2 ———- 把file1,file2打包成name.tar.gz;
z for gzip 格式。

tar -xzvf name.tar.gz ———- 把name.tar.gz解压。

grep

强大的文本搜索工具

grep -i "line" file ———- 搜索file中有”line”的行(i for ignore-case)

grep -e "regex" file ———– 搜索file中符合”regex”的行(e for regular expression)

grep -e "pattern1" -e "pattern2" file -o ———- 输出file中符合pattern1和pattern2的行(o for just print output)

grep -l "line" file1 file2 ... ———- 在file1, file2 …中搜索含有”line”的文件(l for just list file name)

grep "line" . -r -l ———- 在当前目录中找出含有”line”的文件(. -r for recursive search in current directory)

netstat

查看Linux中网络系统状态信息

netstat -a ———– 列出所有的端口。(a for all)

netstat -at ———- 列出所有tcp端口。(t for tcp)

netstat -au ———- 列出所有udp端口。(u for udp)

netstat -al ———– 列出所有监听端口。(l for listening)

netstat -nt ———— 以ip形式(而不是域名服务器)列出tcp协议。(n for numeric);
awk '{print $3}' ———– 打印第三个字段 (外部地址).

ssh

openssh套件中的客户端连接工具

ssh user@ip ———– 以user身份登陆ip服务器。

scp

加密的方式在本地主机和远程主机之间复制文件

scp user@ip:dir1/file dir2 ———- 以user登陆ip服务器后, 下载ip:dir1/fie到本地dir2.

scp user@ip:dir1 dir2 ———- 以user登陆ip服务器后, 下载目录ip:dir1到本地dir2.(r for recursive copy)

scp -r file user@ip:dir1 ———- 以user登陆ip服务器后, 上传file到ip:dir1.

scp -r dir2 user@ip:dir1 ———- 以user登陆ip服务器后, 上传目录dir2到ip:dir1.

xargs

给其他命令传递参数的一个过滤器

xargs 之所以能用到这个命令，关键是由于很多命令不支持 | 管道来传递参数，而日常工作中有有这个必要，所以就有了xargs命令.

xargs -dX ———— 以X作为 deliminator.(将X转换为空格)

xargs -n2 ———— 每次输入两个。(以空格为每个的deliminator)

xargs -I{} cmd {} ———- 将{}作为参数输入的位置。

cut

连接文件并打印到标准输出设备上

cut -f 1 file ——— 提取第一个字段。(f for field)

cut -c 2-4 file ———- 提取2-4个字符。(c for character)

cut -b -5 file ———- 提取前5个字节。(b for byte)

awk

文本和数据进行处理的编程语言

awk 'BEGIN{cmd1}{cmd1}END{cmd1}' ———- 执行cmd1, 逐行执行cmd2, 最后执行cmd3.

NF NR $0 $1 $2 $3 ———– 在awk中分别代表： 字段数，行数，第一个字段，第二个字段，第三个字段。

$NF $(NF-1) ———- 在awk中分别代表最后一个字段，倒数第二个字段。

awk BEGIN{sum=0}{sum+=1}END{print sum} ———- 定义累加变量。

sed

功能强大的流式文本编辑器

sed 's/word1/word2/' file ———- 替换file每行中的word1第一次出现为word2.

's/word1/word2/g' ————- 替换file每行中的全部word1为word2.
's/word1/word2/2g' ————- 替换file每行中的第二个到最后一个word1为word2.

sed '1,2d' file ———- 删除文件的1,2行。

'/pattern/d' ———- 删除file中match pattern的行。

sed -n /pattern/p' file ———– 打印file中符合pattern的行。

-n '1,2p' file ———- 打印file第一行和第二行。
